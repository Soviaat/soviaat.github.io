So, what is Statify?

Statify basically lets you visualize your in-game statistics inside a Google Spreadsheet. It provides Pie Charts of important stats and some not so important (silly) stats.

This mod uploads your statistics into your Google Spreadsheets every 2 minutes.


Charts from luckeY's Statify spreadsheet.

What mods are needed for Statify?

Currently the mod is only compatible with Fabric API and its Loader. I didn't plan to make a Forge version, but if there is enough demand for it, then I'll think about it.

The required Fabric versions for the current release of Statify are:

Name
Version
Fabric API
0.110.5+1.21.4
Fabric Loader
0.16.9
